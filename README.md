# myops

## 开发环境
-----------------------------------------
Python: 2.7
Django: 1.8.5
MySQL: >= 5.1

## 主要功能

### 主机

- In/Out消息统计
- Topic增量数据排名
- Consumer延迟实时排名
- 可查看每个Topic Partition的Logsize Offsets Lag数据
- 实时获取指定Group的Logsize Offsets Lag
- 可定制的历史数据存储时间
